# WeatherApp: Прогноз погоды на Python

Простое и красивое десктопное приложение для просмотра погоды, написанное на Python с использованием библиотек `tkinter` и `requests`.

Приложение получает данные с бесплатного API **Open-Meteo**, автоматически определяет координаты города и показывает текущую погоду вместе с прогнозом на 3 дня.

![Пример интерфейса](assets/screenshot.png) <!-- Замени на путь к своему скриншоту, если есть -->

## ✨ Особенности

*   **Графический интерфейс:** Современный вид благодаря темам `ttkthemes` (если установлены).
*   **Асинхронность:** Запросы к серверу выполняются в отдельном потоке (`threading`), поэтому интерфейс не зависает во время загрузки данных.
*   **Обработка ошибок:** Корректные сообщения при неверном названии города, проблемах с сетью или таймауте.
*   **Бесплатный API:** Не требует регистрации ключей или API-токенов.
*   **Кроссплатформенность:** Работает на Windows, macOS и Linux.
